## useful functions
* eccak256 
```
built in, which is a version of SHA3. A hash function basically maps an input string into a random 256-bit hexidecimal number.
example:
//b1f078126895a1424524de5321b339ab00408010b7cf0e6ed451514981e58aa9
keccak256("aaaac");
```
